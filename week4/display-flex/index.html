display: flex;
justify-content: center; /* 가로 중앙 */
align-items: center; /* 세로 중앙 */

📌 예제 4: 반응형 카드 그리드

flex: 1 1 calc(33.333% - 14px)로 반응형 3단 레이아웃 구현

.card {
  flex: 1 1 calc(33.333% - 14px);
  min-width: 250px;
}
📱
카드 1
반응형 디자인으로 화면 크기에 따라 자동으로 레이아웃이 조정됩니다.
🎨
카드 2
모든 카드의 높이가 자동으로 맞춰집니다. align-items: stretch 효과입니다.
⚙️
카드 3
화면이 작아지면 2단, 3단, 1단으로 자동 조정됩니다. 시도해보세요!
