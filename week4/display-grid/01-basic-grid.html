가장 기본적인 그리드 레이아웃입니다. 3개의 열과 3개의 행으로 구성되어 있습니다.

.grid-basic {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  grid-template-rows: repeat(3, 100px);
  gap: 15px;
}

2. 다양한 크기 조합

고정 크기와 유동 크기를 조합한 그리드입니다. 첫 번째 열은 200px, 두 번째는 남은 공간, 세 번째는 100px입니다.

100px * 60px
.grid-mixed {
  display: grid;
  grid-template-columns: 200px 1fr 100px;
  grid-template-rows: 80px 1fr 60px;
  gap: 20px;
}

3. fr 단위 (Fraction) 사용

fr 단위를 사용하여 비율로 크기를 지정합니다. 1fr : 2fr : 1fr 비율로 설정되어 있습니다.

1fr
2fr (2배 넓음)
1fr
.grid-fraction {
  display: grid;
  grid-template-columns: 1fr 2fr 1fr;
  gap: 15px;
}

4. auto-fit 사용 (반응형)

auto-fit을 사용하여 화면 크기에 따라 자동으로 열 개수가 조정됩니다. 각 항목은 최소 200px, 최대 1fr입니다.
