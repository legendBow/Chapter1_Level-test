🚀 고급 기법 - CSS Grid

CSS Grid 고급 기법들

CSS Grid의 고급 기능들을 활용하여 더욱 정교하고 유연한 레이아웃을 만들어보겠습니다.

1. minmax() 함수

최소값과 최대값을 설정하여 유연한 크기 조정이 가능합니다. auto-fit과 함께 사용하면 반응형 그리드를 만들 수 있습니다.

최소 200px
유동 크기
자동 조정
반응형
minmax()
함수 사용
.minmax-grid {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
  gap: 20px;
}

/* minmax(최소값, 최대값) */
/* minmax(200px, 1fr) = 최소 200px, 최대 1fr */
/* minmax(100px, auto) = 최소 100px, 최대 내용에 맞춤 */

2. clamp() 함수

최소값, 선호값, 최대값을 설정하여 반응형 크기를 만듭니다. 뷰포트 크기에 따라 자동으로 조정됩니다.

clamp(200px, 50vw, 500px)
1fr (유동)
.clamp-grid {
  display: grid;
  grid-template-columns: clamp(200px, 50vw, 500px) 1fr;
  gap: 20px;
}

/* clamp(최소값, 선호값, 최대값) */
/* clamp(200px, 50vw, 500px) = 최소 200px, 선호 50vw, 최대 500px */

3. fit-content() 함수

내용에 맞춰 크기를 조정하되, 최대값을 제한합니다. 텍스트나 이미지의 크기에 따라 자동으로 조정됩니다.

fit-content(300px)
1fr (유동)
.fit-content-grid {
  display: grid;
  grid-template-columns: fit-content(300px) 1fr;
  gap: 20px;
}

/* fit-content(최대값) */
/* fit-content(300px) = 내용에 맞춤, 최대 300px */

4. Named Grid Lines (이름 지정)

그리드 라인에 이름을 지정하여 더 읽기 쉬운 코드를 작성할 수 있습니다.

사이드바
헤더
메인 콘텐츠
푸터
.named-lines-grid {
  display: grid;
  grid-template-columns: [sidebar-start] 200px [sidebar-end main-start] 1fr [main-end];
  grid-template-rows: [header-start] 80px [header-end content-start] 1fr [content-end footer-start] 60px [footer-end];
  gap: 20px;
}

/* 이름을 사용한 배치 */
.sidebar {
  grid-column: sidebar-start / sidebar-end;
  grid-row: header-start / content-end;
}

5. auto-flow dense

빈 공간을 자동으로 채우는 dense 모드입니다. 뒤의 아이템이 앞의 빈 공간을 채웁니다.

.dense-grid {
  display: grid;
  grid-template-columns: repeat(4, 1fr);
  grid-auto-flow: row dense;
  gap: 15px;
}

/* dense 모드로 빈 공간 자동 채우기 */
.item:nth-child(1) { grid-column: span 2; }
.item:nth-child(2) { grid-column: span 1; }

/* 컨테이너 전체 정렬 */
.container {
  justify-items: center; /* 수평 정렬 */
  align-items: center; /* 수직 정렬 */
  place-items: center; /* 수평+수직 동시 */
}

/* 개별 아이템 정렬 */
.item {
  justify-self: center; /* 개별 수평 정렬 */
  align-self: center; /* 개별 수직 정렬 */
  place-self: center; /* 개별 수평+수직 동시 */
}

7. 개별 아이템 정렬

고급 기법 활용 팁

/* 1. 반응형 그리드 조합 */
grid-template-columns: repeat(auto-fit, minmax(clamp(200px, 30vw, 400px), 1fr));

/* 2. 복잡한 레이아웃을 위한 Named Lines */
grid-template-columns: [sidebar-start] 200px [sidebar-end main-start] 1fr [main-end];

/* 3. 정렬과 크기 조정 조합 */
place-items: center;
grid-template-columns: fit-content(300px) 1fr;

/* 4. 중첩 그리드로 복잡한 구조 */
.parent { display: grid; }
.child { display: grid; }
