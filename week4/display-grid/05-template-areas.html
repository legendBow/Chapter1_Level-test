Grid Template Areas 활용

Grid Template Areas를 사용하여 복잡한 레이아웃을 직관적으로 만들 수 있습니다. ASCII 아트 형식으로 영역을 정의합니다.

전형적인 웹사이트 레이아웃입니다. 헤더, 네비게이션, 메인 콘텐츠, 사이드바, 푸터로 구성됩니다.

메인 콘텐츠

이곳은 웹사이트의 주요 콘텐츠가 들어가는 영역입니다. Grid Template Areas를 사용하여 레이아웃을 구성했습니다.

각 영역은 grid-area 속성을 사용하여 이름을 지정하고, grid-template-areas에서 ASCII 아트 형식으로 배치합니다.

사이드바

  • 관련 링크 1
  • 관련 링크 2
  • 관련 링크 3
  • 관련 링크 4
© 2024 웹사이트. All rights reserved.
.website-layout {
  display: grid;
  grid-template-areas:
    "header header header"
    "nav main sidebar"
    "footer footer footer";
  grid-template-columns: 200px 1fr 250px;
  grid-template-rows: 80px 1fr 60px;
  gap: 20px;
}

/* 각 영역에 이름 지정 */
header { grid-area: header; }
nav { grid-area: nav; }
main { grid-area: main; }
aside { grid-area: sidebar; }
footer { grid-area: footer; }

이 레이아웃은 모바일 화면에 최적화되어 있습니다. 모든 요소가 세로로 배치되어 터치하기 쉽고 읽기 편합니다.

모바일 사이드바

모바일에서는 사이드바가 메인 콘텐츠 아래에 배치됩니다.

/* 모바일용 미디어 쿼리 */
@media (max-width: 768px) {
  .website-layout {
    grid-template-areas:
      "header"
      "nav"
      "main"
      "sidebar"
      "footer";
    grid-template-columns: 1fr;
    grid-template-rows: auto auto 1fr auto auto;
  }
}

Template Areas의 장점

/* 1. 직관적인 레이아웃 정의 */
grid-template-areas:
  "header header header"
  "nav main sidebar"
  "footer footer footer";

/* 2. 쉬운 반응형 변경 */
@media (max-width: 768px) {
  grid-template-areas:
    "header"
    "nav"
    "main"
    "sidebar"
    "footer";
}

/* 3. 명확한 영역 구분 */
.header { grid-area: header; }
.main { grid-area: main; }
.sidebar { grid-area: sidebar; }
