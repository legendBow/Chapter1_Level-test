🖼️ 반응형 이미지 갤러리

다양한 갤러리 스타일

CSS Grid를 사용하여 다양한 스타일의 반응형 이미지 갤러리를 만들어보겠습니다.

1. 기본 반응형 갤러리 (auto-fit)

화면 크기에 따라 자동으로 열 개수가 조정되는 갤러리입니다. 각 이미지는 최소 250px, 최대 1fr 크기입니다.

해변의 일몰

.gallery {
  display: grid;
  gap: 15px;
  grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
}

.gallery-item {
  aspect-ratio /* 종횡비 */ : 4/3;
  border-radius: 12px;
  overflow: hidden;
  box-shadow: 0 8px 25px rgba(0,0,0,0.15);
  transition: transform 0.3s ease;
}

.gallery-masonry {
  display: grid;
  gap: 15px;
  grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
  grid-auto-rows: 10px;

/* 각 아이템의 높이를 다르게 설정 */
.masonry-item:nth-child(1) { grid-row: span 30; }
.masonry-item:nth-child(2) { grid-row: span 25; }

3. 고정 크기 갤러리 (auto-fill)

auto-fill을 사용하여 빈 공간이 생기는 갤러리입니다. 각 이미지는 고정된 크기를 가집니다.

.gallery-fixed {
  display: grid;
  gap: 20px;
  grid-template-columns: repeat(auto-fill, minmax(300px, 1fr));
}

.fixed-item {
  height: 250px;
  border-radius: 12px;
  overflow: hidden;
}

auto-fit vs auto-fill 차이점

1fr은 실제 콘텐츠가 있을 때만 적용됨
max: 1fr 이더라도 빈공간이 생긴다는 건 그 열이 실제로 존재하지만 내용이 없어서 높이가 0으로 남는 것
/* auto-fit: 여유 공간을 기존 열들이 나눠 가짐 */
grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));

/* auto-fill: 빈 열도 만들어짐 */
grid-template-columns: repeat(auto-fill, minmax(250px, 1fr));

/* 예시: 1200px 화면, 250px 기준 */
/* auto-fit: [250px] [250px] [250px] [250px] [200px 확대] */
/* auto-fill: [250px] [250px] [250px] [250px] [빈 공간] */
