2️⃣ Inline 요소 예제

Inline 요소는 필요한 만큼만 너비를 차지하고 같은 줄에 여러 요소를 배치할 수 있습니다.

이 문장에는 Inline 1 Inline 2 Inline 3 와 같은 인라인 요소가 있습니다.

모든 요소가 텍스트처럼 같은 줄에 배치되는 것을 확인하세요.

Inline 요소는 width/height 무시

이 요소는 margin과 padding이 좌우만 적용되고 상하는 무시됩니다.

상하 여백이 시각적으로 보이지 않습니다. 오직 좌우만 작동합니다.

3️⃣ Inline-Block 요소 예제

Inline-block 요소는 같은 줄에 여러 요소를 배치하면서도 크기를 조정할 수 있습니다.

📱 Card 1
높이와 너비 조정 가능
🎨 Card 2
같은 줄에 배치됨
⚡ Card 3
상하 여백도 적용됨

3개의 카드가 같은 줄에 배치되면서도 각각 정확한 크기를 가집니다. 마우스를 올려보세요!

4️⃣ 네비게이션 바 (Inline-Block 활용)

실무에서 자주 사용되는 네비게이션 메뉴 구현:

🏠 홈 📖 소개 🛍️ 제품 💬 블로그 📞 연락처

메뉴 항목들을 마우스로 가리켜보세요. Inline-block으로 수평 배치되면서 padding과 background가 정상 작동합니다.

5️⃣ 버튼 그룹 (Inline-Block 활용)

여러 버튼을 한 줄에 배치:

버튼들을 마우스로 가리켜보세요. Display를 이용한 정렬입니다.

6️⃣ 배지/태그 (Inline-Block 활용)

블로그 포스트나 상품에 사용되는 배지 디자인:

CSS 초급 필독 추천

JavaScript 중급 웹 개발

배지들이 텍스트처럼 자동으로 줄바꿈됩니다.

7️⃣ 폼 디자인 (Block 활용)

Block과 Inline을 혼합한 폼 디자인:

실명으로 입력해주세요
유효한 이메일 주소를 입력해주세요
하이픈(-)을 포함하여 입력해주세요

각 입력 필드는 Block 요소로 전체 너비를 차지합니다. 레이블과 입력 필드가 세로로 정렬됩니다.

📚 핵심 정리

속성 특징 사용 예
Block 전체 너비 차지, 새 줄 시작 페이지 레이아웃, 폼
Inline 필요한 만큼만 너비, 같은 줄 텍스트 강조, 링크
Inline-Block 크기 조정 + 같은 줄 배치 카드, 배지, 버튼
